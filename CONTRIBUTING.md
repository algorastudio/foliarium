# Contribuire a Foliarium

Grazie per il tuo interesse nel contribuire a Foliarium! Questo documento descrive le linee guida per contribuire al progetto.

## Come contribuire

### Segnalare bug o problemi

1. Verifica che il problema non sia già stato segnalato nelle [Issues](https://github.com/algorastudio/foliarium/issues)
2. Crea una nuova Issue descrivendo:
   - Il comportamento atteso
   - Il comportamento riscontrato
   - I passi per riprodurre il problema
   - Il sistema operativo e la versione di Python utilizzata

### Proporre nuove funzionalità

1. Apri una Issue con il tag `enhancement`
2. Descrivi la funzionalità proposta e il suo valore per gli utenti
3. Attendi un feedback dal team prima di iniziare lo sviluppo

### Inviare modifiche al codice

1. Fai un fork del repository
2. Crea un branch per la tua modifica: `git checkout -b feature/nome-funzionalita`
3. Scrivi codice chiaro e commentato
4. Assicurati che i test passino
5. Invia una Pull Request descrivendo le modifiche effettuate

## Standard di codice

- Segui le convenzioni PEP 8 per Python
- Usa docstring per le funzioni pubbliche
- Scrivi commenti in italiano o inglese
- Mantieni le funzioni brevi e con una sola responsabilità

## Licenza
